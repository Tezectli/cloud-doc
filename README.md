# 云文档桌面应用
#### 使用electron和React搭建的桌面应用，实现了自动云同步的功能，可以自行导入Markdown文件进行集中整理 
#### 也可以在应用中新建md项目

> 努力完善中……


## 技术选型

客户端：electron + react 

服务端：Node.js + 七牛云

## 安装运行

* #### 第一步
    * 克隆项目到本地`https://github.com/Tezectli/cloud-doc.git`
* #### 第二步
    * `npm install`安装依赖
* #### 第三步
    * `npm run dev`运行项目
